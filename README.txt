# Financial Chatbot

This chatbot answers predefined financial queries about Apple and Samsung using Python and pandas.

## Supported Queries:
- What is the total revenue for Apple in 2023?
- What is the total revenue for Samsung in 2023?
- How has Apple’s net income changed from 2022 to 2023?
- How has Samsung’s net income changed from 2022 to 2023?
- What’s the overall revenue growth of Apple?

## How it works:
- Uses a pandas DataFrame to store and analyze data
- Uses `if-elif-else` logic to match exact queries
- Accepts user input via the command line

## Limitations:
- Only supports 5 predefined queries
- Does not handle typos or flexible input
- Basic CLI chatbot only (no web or GUI)
